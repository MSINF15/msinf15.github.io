Ceci est un exemple de site statique. Son contenu est écrit en dur et n'est pas généré à la volé par le serveur avec du contenu dit «dynamique». Cela signifie également que le contenu ne varie pas en fonction de la personne qui vient visiter ce site.

Pour rendre cette page visuellement plus plaisante, on peut utiliser le framework Bootstrap qui définit déjà pleins de classes que l'on peut simplement réutiliser dans notre page HTML pour en modifier l'apparence. Bootstrap définit également un certain nombre de fonctions Javascript permettant de rendre la page plus interactive. Exemple avec cet élément de classe accordion :

This is the first item's accordion body. It is shown by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.
