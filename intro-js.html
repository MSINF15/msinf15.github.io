Chargement d'un fichier Javascript

Pour charger un fichier Javascript dans une page HTML, on peut insérer la balise suivante dans le head : 

                    
<head>
    <script src="./chemin/vers/le/script.js"></script>
</head>

Éléments de base

Voici un bout de code en Javascript qui montre comment déclarer une variable, créer une instruction conditionnelle, et créer une boucle for. 

                    
  let entier = 0;
  let chaine = "Bonjour";
  let valeurBooleenne = true || false;

  if (valeurBooleenne) {
    alert(entier);
    alert(chaine);
  } else if (valeurBooleenne) {
    // On ne devrait jamais arriver là
  } else {
    // Ici non plus
  }

  let message = "";
  for (let i = 0; i < 10; i++) {
    message += i;
  }
  alert(message);

Exemple "onclick"

Voici un exemple de bouton en HTML, ainsi que le code correspondant :

                    
<button class="btn btn-primary" onclick="buttonClicked();">Click me!</button>
<!-- L'attribut onclick="buttonClicked();" signifie que :
  -- Lorsque le bouton est cliqué, une fonction Javascript intitulée
  -- "buttonClicked" est appelée. Celle-ci doit se trouver dans les fichiers
  -- Javascript chargés par cette page HTML. -->

Voici le contenu de la fonction buttonClicked : 

                
function buttonClicked() {
    alert("Le bouton a été cliqué.");
}

Manipulation d'éléments HTML

Il est possible d'ajouter ou de modifier le contenu d'une page HTML avec Javascript. 

                    
function modificationDOM() {
  let snippets = document.getElementsByClassName('code-prism');
  for (let i = 0; i < snippets.length; i++) {
    snippets.item(i).style.background = "yellow";
  }
}

function ajoutDOM() {
  let element = document.getElementById('ajout');
  element.innerHTML = "<p class=\"display-6\">Texte rajouté</p>"
    + "<div class=\"row\">"
    + "<p class=\"text-start\">Ceci est un exemple de texte ajouté par Javascript.</p>"
    + "</div>";
}
